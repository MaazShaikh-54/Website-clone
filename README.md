# Penta Freight Website Clone
React + Tailwind CSS clone built as an interview assignment.

## Features
- Responsive mobile navbar with slide-in animation and hamburger-to-X transition
- Active link highlighting (orange) with dropdown for Services and PentaKUHL
- Cookie consent banner
- Video hero section
- Achievements section with animated counters
- India & USA office footer
- Reviews carousel, certifications scroll, awards showcase

## Tech Stack
- React
- Vite
- Tailwind CSS
